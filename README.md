# Actividad proga III

### ¿Para qué sirve el archivo package.json?
El archivo package.json contiene la informacion de nuestro proyecto como lo es el nombre, version y configuraciones para levantarlo, junto con todas dependecias y librerias que utilizamos.  

### ¿Qué diferencia notaron entre ejecutar un archivo directamente con Node.js y levantar el servidor de React?
La diferencia que note es que en el servidor de node cada vez que aplicamos cambios debemos volve a levantar el proyecto, en cambio al servidor de react escucha los cambios y re-renderiza al mismo tiempo que desarrollamos
